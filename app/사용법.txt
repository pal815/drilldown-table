==========================================================
  드릴다운 표 생성기 — 사용법
  (계층형 ㄱ자/코너헤더 표를 Excel · Word · PowerPoint로 자동 생성)
==========================================================

설치가 필요 없습니다. 파이썬·언어모델 없이 바로 씁니다.
이 폴더(드릴다운표생성기)를 통째로 두고 아래 EXE를 더블클릭하세요.

----------------------------------------------------------
1. 실행하기
----------------------------------------------------------
  ▸ 「드릴다운표생성기.exe」 더블클릭.

  ▸ 처음 실행 시 파란 SmartScreen 경고
    ("Windows의 PC 보호" / "알 수 없는 게시자")가 뜰 수 있습니다.
    → [추가 정보] 클릭 → [실행] 버튼.
       (서명되지 않은 사내 프로그램이라 뜨는 정상 경고입니다.)

  ▸ 메일/다운로드로 받은 경우 차단되어 안 열릴 수 있습니다.
    → 받은 ZIP 파일에서 [마우스 오른쪽] → [속성]
      → 아래쪽 "차단 해제(Unblock)" 체크 → [확인]
      → 그 다음 압축을 푸세요. (압축 푼 뒤엔 EXE가 정상 실행)

----------------------------------------------------------
2. 사용 순서
----------------------------------------------------------
  (1) 입력 데이터 넣기 — 둘 중 편한 방법
       · 엑셀 파일 첨부:  [엑셀 파일 선택] 버튼
       · 복사-붙여넣기:   엑셀에서 표 영역을 복사(Ctrl+C) →
                          앱의 [클립보드에서 가져오기] 버튼
  (2) 방향 선택:  행 기준 / 열 기준 / 행+열(교차표)
  (3) 테마 선택:  색상 / 회색 / 흑백
  (4) 형식 선택:  Excel / Word / PowerPoint (복수 선택 가능)
  (5) [표 만들기] → 완료되면 결과 파일이 열립니다.

  · 입력 형식이 헷갈리면 화면의 ⓘ(입력 형식 예시) 버튼을 누르세요.
  · 각 방향/테마 버튼에 마우스를 올리면 결과 미리보기가 뜹니다.

----------------------------------------------------------
3. 입력 표 작성법 (★다단계 지원)
----------------------------------------------------------
  왼쪽에 "분류 라벨 열"을, 그 오른쪽에 "값(매출·이익 등)"을 적습니다.

  ● 2단계 예시
       항목      세부    매출   영업이익
       A사업부    계     1200    180
                 Q1      280     38
                 Q2      300     42
       ...

  ● 3·4단계 예시 — 라벨 열을 단계 수만큼 늘리면 자동 인식
       사업부    지역    분기    매출   영업이익
       A사업부                  1200    180
                국내            700     120
                        Q1      300      55
                        Q2      400      65
                해외            500      60
       B사업부                  1000    230
                Q1              480     110
                ...

  팁:
   · 합계 줄("계")은 적어도 되고, 생략하면 자동으로 합산해 줍니다.
   · 가지마다 깊이가 달라도 됩니다(예: A는 3단계, B는 2단계).
   · 맨 위 제목·단위 줄은 선택 사항(비워도 됩니다).
   · "열 기준"을 고르면 위 분류가 가로로 펼쳐진 표가 됩니다.

----------------------------------------------------------
4. 결과 파일 위치
----------------------------------------------------------
  기본 저장 위치는 바탕화면입니다.
  (엑셀 파일을 첨부한 경우엔 그 파일과 같은 폴더에 저장)
  파일명 예: 드릴다운표_row_color.xlsx

----------------------------------------------------------
5. 같이 들어 있는 「입력양식.xlsx」
----------------------------------------------------------
  바로 채워 쓰는 빈 양식입니다. 시트 두 개:
    · "행·열"      : 행 기준 / 열 기준용
    · "행+열(교차)" : 2차원 교차표용
  여기에 값만 채워 저장한 뒤 앱에 첨부하면 됩니다.

----------------------------------------------------------
6. 문제가 생기면
----------------------------------------------------------
  · 엑셀에 시트가 여러 개이거나 DRM(보안)이 걸려 있어도,
    앱이 자동으로 Excel을 통해 읽어옵니다(해당 PC에 Excel 필요).
  · 회사 PC가 잠겨 EXE 실행이 막히는 경우, 별도 웹 버전
    (drilldown_web.html)을 브라우저에서 쓸 수 있습니다(IT 문의).
